# **App Name**: WhatsAi

## Core Features:

- Onboarding Flow: Guide users through a mandatory onboarding process, collecting brand information, contact details, and initial AI configurations. Automatically generate Open Graph Meta Tags to improve credibility and click-through rates.
- Contact Management: Provide a system for organizing, editing, and analyzing contacts. Users can view, import, edit, and categorize contacts. Contacts and metadata are stored in Firestore.
- Chat Interface: Include a chat environment for attendants with real-time message updates, AI activation/deactivation, inline contact editing and clear isolation.
- AI-Powered Chat Summarization: Automatically generate a Structured AI Summary at the end of each conversation and maintain a persistent memory per client for continuity in future sessions.
- AI-Driven Profile Enrichment: Use LLM intelligence to analyze chat content and suggest new interests, adjusted contact categories, opportunity insights and internal notes. Suggestions are saved and can be accepted or dismissed by the attendant.
- Intelligent Follow-Up Suggestions: Use AI to generate personalized email drafts, WhatsApp messages, follow-up recommendations and pre-created events for Google Calendar. The LLM acts as a tool, using customer information responsibly and adapting outputs to the brand’s tone and rules.
- PWA Chat for Clients: Offer a lightweight PWA chat route that displays brand OG metadata, requests only the client’s phone number, provides real-time messaging, supports offline fallback and initiates an automatic AI greeting upon entry.

## Style Guidelines:

- Primary color: Sea Green (#30D155) - vibrant and modern, ideal for primary actions.
- Background color: Light Green (#E5F8E8) - soft, airy, and unobtrusive.
- Accent color: Forest Green (#6ACC7C) - subtle highlights for interactive elements.
- PT Sans — main UI and body font; modern, warm, and readable.
- Source Code Pro — used for code or technical text.
- Minimalist iconography
- Clean, responsive layout
